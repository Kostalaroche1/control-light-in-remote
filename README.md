# control-light-in-remote
this project IOT control light in remote by using wifi and esp32-cam, it is possible to use it with ESP32.
the system create a server and take IP in network where is connected to allow everyone who can use app by connecting with this IP address.
after that you can control light in remote. so the app content just height button wich match both turn on and turn off with label indicate that light is ON or OFF. this doesn't involve internet, you can use it by local network or wifi.
